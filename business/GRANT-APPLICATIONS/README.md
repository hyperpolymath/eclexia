# Grant Applications

This directory stores active and submitted grant applications for Eclexia.

## Recommended Structure
- `YYYY-MM-DD_<funder>_<program>.md` for each application draft.
- `attachments/` for supporting material (budget, letters, timelines).
- `submitted/` for immutable final submissions.

## Contents
- `TEMPLATE.md`: reusable full grant application template.
- `CHECKLIST.md`: pre-submission quality checklist.
