# Grant Submission Checklist

## Compliance
- [ ] Eligibility confirmed.
- [ ] Word/page limits respected.
- [ ] Required attachments included.
- [ ] Budget format matches funder template.

## Technical Quality
- [ ] Objectives are measurable.
- [ ] Milestones map to deliverables.
- [ ] Risks include explicit mitigations.
- [ ] Evaluation metrics are concrete.

## Narrative Quality
- [ ] Problem statement is specific and evidence-backed.
- [ ] Differentiation is clear.
- [ ] Impact claims are realistic and testable.
- [ ] Sustainability plan is credible.

## Review Process
- [ ] Internal technical review complete.
- [ ] External reviewer feedback incorporated.
- [ ] Final owner sign-off complete.
- [ ] Submission confirmation archived.
