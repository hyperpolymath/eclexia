# Glossary

- Economics-as-Code: programming model where resource and optimization constraints are first-class
- Shadow price: marginal value/cost signal used for resource tradeoffs
- Adaptive function: function with multiple solutions selected by constraints/objectives
- HIR: high-level intermediate representation
- MIR: mid-level intermediate representation
- Conformance test: test that validates language behavior against intended semantics
- FFI: foreign function interface between Eclexia and external/native code
- ABI: binary contract for data layout and call conventions
