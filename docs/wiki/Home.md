# Eclexia Wiki Home

Eclexia is a programming language project focused on Economics-as-Code: making energy, latency, memory, and carbon constraints first-class in software.

- I am new and non-technical: [For Laypeople](For-Laypeople.md)

- Status: Alpha
- Core compiler pipeline: functional
- Runtime/resource model: functional with ongoing hardening
- Formal proofs: active, with some remaining admitted theorems
