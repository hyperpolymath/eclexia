== In Progress

=== v0.2.0 - Core Development (In Progress)

- [x] 25-crate Rust compiler architecture
- [x] Lexer (logos, 893 lines, 95+ tokens)
- [x] Parser (Pratt, 3106 lines, 32 valid + 19 invalid conformance tests)
- [x] Type checker (Robinson unification, 2210 lines, dimensional types)
- [x] HIR and MIR lowering
- [x] Bytecode codegen and stack-based VM (934 lines)
- [x] Tree-walking interpreter with builtins
- [x] .eclb binary bytecode format
- [x] LSP server, formatter, linter, debugger
- [x] 271 lib tests passing, 0 conformance skips
- [ ] Wire reactive compiler crates (salsa DB, modules, effects) into CLI
- [ ] Reduce unwrap() calls in parser/interpreter
- [ ] Link builtins into bytecode build path
- [ ] Real codegen for backend crates (Cranelift, LLVM, WASM)
