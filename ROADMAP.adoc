== In Progress

=== v0.2.0 - Core Development (In Progress â€” ~75% Complete)

==== Compiler Pipeline (Complete)
- [x] 25-crate Rust compiler architecture
- [x] Lexer (logos, 893 lines, 95+ tokens)
- [x] Parser (Pratt, 3106 lines, 32 valid + 19 invalid conformance tests)
- [x] Type checker (Robinson unification, 2210 lines, dimensional types)

- [x] MIR with optimizations (constant propagation, dead code elimination, block inlining)
- [x] Bytecode codegen with concurrency builtins
- [x] Stack-based VM (934 lines, 18 unit tests)
- [x] Tree-walking interpreter with tokio concurrency
- [x] .eclb binary bytecode format

- [x] Shadow prices with real defaults and LP duality pricing (8+ tests)
- [x] Resource tracking: energy, time, memory, carbon
- [x] Adaptive decision engine with budget enforcement (7 tests)
- [x] Scheduler: shadow-price-aware defer/reject/run (4 tests)
- [x] Profiler: wall-clock + energy/carbon + RSS memory tracking (6 tests)
- [x] Carbon monitor: grid intensity Green/Yellow/Red (7 tests)
